## Description

Realworld backend implementation using TypeScript + NestJS

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# watch mode
$ yarn start

# production mode
$ yarn run start:prod
```

## Work with DataBase 

```bash
# drop all tables 
$ yarn db:drop

# create migration
$ yarn db:create

# applying migrations
$ yarn db:migrate

# load seeds
$ yarn db:seed
```
